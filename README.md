# Air_Quality_Index_-AQI-_Prediction

## 📌 Project Overview

This project focuses on analyzing air pollution data to:

📊 Predict AQI values (Regression)
🏷️ Classify AQI into categories (Classification)

The goal is to understand how different machine learning algorithms perform on real-world environmental data.

## 🎯 Objectives
Build regression models to predict exact AQI values
Build classification models to predict AQI buckets
Compare model performance and extract meaningful insights

## 🧠 Algorithms Used

🔹 Regression Models
K-Nearest Neighbors (KNN Regressor)
Decision Tree Regressor

🔹 Classification Models
K-Nearest Neighbors (KNN Classifier)
Decision Tree Classifier
Gaussian Naive Bayes

## 📊 Features Used
PM2.5
NO
NO2
NOx
CO
SO2
O3

(All features are continuous numerical values related to air pollutants)

## ⚙️ Tech Stack
Python
Pandas
NumPy
Scikit-learn
Matplotlib

## 🔄 Workflow

Data Cleaning & Preprocessing
Feature Selection
Train-Test Split
Model Building
Model Evaluation
Performance Comparison

## 💡 Key Insights

🔴 KNN tends to overfit with high training accuracy
🟢 Decision Tree provides balanced and reliable performance
🔵 Naive Bayes struggles due to data distribution assumptions

## 🌱 Real-World Impact

Predict air pollution levels in advance
Help authorities take preventive measures
Enable early health warnings for the public
